To clone a specific branch from a shared repository using Git Bash
cd path/to/your/directory

Clone the Repository with a Specific Branch:
git clone -b branch-name repository-url
For example, if you want to clone a branch named "feature-branch" from a repository with the URL "https://github.com/example/repo.git," the command would look like this:

git clone -b feature-branch https://github.com/example/repo.git

Navigate into the Cloned Repository:
Change into the directory created by the clone operation.

cd repo
Now, you have successfully cloned a specific branch from the shared repository. You can work on the code in the cloned branch on your local machine. If you need to switch to a different branch, you can use the git checkout command:

git checkout other-branch-name
Replace other-branch-name with the name of the branch you want to switch to.

Keep in mind that you can also use the git branch -a command to list all branches (local and remote) in the cloned repository, and then switch between them as needed.

--------------------------

Create a New Branch:
Create a new branch locally using the following commands:

git checkout -b new-branch-name
